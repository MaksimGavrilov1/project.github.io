JavaScript

JavaScript - предназначен для написания сценариев
для активных HTML-страниц. С помощью JavaScript можно
динамически изменять текст загружаемого HTML-документа
и реагировать на события, связанные с действиями посетителя
или изменениями состоятия документа или окна.
JavaScript не предназначен для создания автономных
приложений. Программа на JavaScript встраивается непосредственно
в исходный текст HTML-документа и интерпретируется
брaузером по мере загрузки этого документа.
